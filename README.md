## Setup

**1. Repo clonen**
```bash
git clone <url>
cd de-challenge
```

**2. Dependencies installieren**
```bash
pip install -r requirements.txt
```

**3. Notebook öffnen und ausführen**
```bash
jupyter notebook main.ipynb
```
